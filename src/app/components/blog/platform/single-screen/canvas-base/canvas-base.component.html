Platform game: Canvas

Gli strumenti che ci serviranno per creare il nostro platform game

Iniziamo strutturando il nostro progetto, creando i file html, css e js con le impostazioni base.

Potremmo mettere tutto in un unico file html, ma è sempre meglio tenerli separati, così che se il progetto dovesse allargarsi non sorgano problemi di leggibilità e debugging.

HTML

Creiamo il file index.html che ha una sua struttura base, l’unica novità è il tag <canvas> che è la nostra “tela digitale” dove andremo a dipingere il nostro capolavoro.

“Tela? Ma cosa siamo pittori?” No, non sono impazzito, o per lo meno, non del tutto.

Ho parlato di tela digitale perché il canvas ( che in inglese significa Tela ) è un elemento html che ci permetterà di disegnare e gestire il nostro gioco interamente con Javascript.

Il canvas è nato per sostituire una tecnologia, “Adobe Flash”, un programma di terze parti che serviva per la gestione di video e giochi vettoriali, aveva un notevole successo per le tante animazioni che conteneva al suo interno,ma, aveva seri problemi di sicurezza quindi si è deciso di rimpiazzarlo con un elemento html nativo.

Dopo questa postilla, torniamo a noi, dunque quello che vedete nel file Index è tutto l’html che ci serve per il nostro progetto: un tag <canvas> con identificativo ( Id ) per catturarlo con javascript. con tanta fantasia, lo chiameremo “myCanvas”.


CSS

Dopo aver impostato la struttura del nostro progetto, dobbiamo dedicarci allo stile e per farlo, dobbiamo creare un file style.css

Nel body abbiamo aggiunto il modulo flaxbox, “display: flex;“ poi, abbiamo impostato il layout, mettendo il canvas al centro ,”align-items: center;“e “justify-content: center;“ successivamente abbiamo dato un’altezza, una larghezza e uno spessore, diversamente , non si vedrebbe a video.

E questo, è tutto ciò che ci serve per il caso.

COME FUNZIONA IL CANVAS?

Bella domanda, bravi!

Analizziamo come funziona il canvas, così da capirlo e padroneggiarlo, visto che sarà un tassello fondamentale per quasi tutti i giochi che andremo a creare.

Dobbiamo immaginarci il canvas come una griglia di pixel che parte dal punto di origine O e si sviluppa sull’asse delle X in larghezza e sull’asse delle Y in lunghezza.
